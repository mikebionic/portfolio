# Portfolio — Mike (Muhammetberdi) Jepbarov

Personal portfolio website showcasing software engineering projects, technical articles, volunteering, electronics, and robotics work.

## Sections

- **Main** — Introduction, avatar, CV download
- **Contact** — Social links and contact info
- **Web/Mobile** — Featured project carousel + gallery of web and mobile projects
- **Volunteering** — Community contributions, speaking engagements, mentoring
- **Articles** — 21 technical articles published on Medium
- **Electronics** — Electronic project showcases
- **Robotics** — Robotics club projects

## Tech Stack

- HTML5, CSS3, vanilla JavaScript
- Bootstrap 5
- Swiper.js (carousels)
- Magnific Popup (video lightboxes)

## Running Locally

Open `index.html` in a browser. No build step required.
